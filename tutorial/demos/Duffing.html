def duff_osc2(x, v, params):
    omega = params['omega']
    t = params['cur_time']
    return -x-.1*x**3-.1*v+1*sin(omega*t)

The break is an indicative of a break in the branch and is actually a result of the solution being unstable. Not the system, but the solution. By that we mean that while this is considered a solution, it isnât one that will actually continue in a real situation and another solution will necessarily be found.

A simple solution is to change the starting guess to be away from the solution and see if it finds another one. Indeed that happens.

def duff_amp_resid(a):
    return (mu**2+(sigma-3/8*alpha/omega_0*a**2)**2)*a**2-(k**2)/4/omega_0**2
